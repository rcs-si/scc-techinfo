Running the script finds the intel and amd .com links on the tech summary page.
We then need to do some manual processing to match to the cpu names from the csvs.

curl -s https://www.bu.edu/tech/support/research/computing-resources/tech-summary/ \
| grep -oP '<a href="https?://[^"]*(intel|amd)\.com[^"]*"[^>]*>[^<]+</a>' \
| sed -E 's@<a href="([^"]*)"[^>]*>([^<]+)</a>@    '\''\2'\'': '\''\1'\'',@' \
| sort | uniq \
| awk 'BEGIN{print "{"} {dict[NR]=$0} END{for(i=1;i<=NR;i++){if(i==NR) sub(/,$/,"",dict[i]); print dict[i]} print "}"}'
